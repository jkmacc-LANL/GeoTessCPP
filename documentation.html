Configuration

The shell script environment_variables.sh will deduce the modifications to the user's PATH and LD_LIBRARY_PATH necessary to use GeoTessCPP, GeoTessCShell, GeoTessCPPExamples and GeoTessCExamples. It prints these recommendations to the screen but does not make any modifications to the user's environment.

Compiling the code

The GeoTess package came with precompiled binaries for 64 bit Linux, but the code can be recompiled by running make.

For more information about building the code, see the Installation section of the User's Manual
